INTRODUCTION
------------

* "Hover Effects" module will provides 30+ Hover effects on your
 square and circle images.
 The asset ihover.css is taken from this project: https://github.com/gudh/ihover
 and licensed under the MIT license.

REQUIREMENTS
------------

* This module does not require any dependencies.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. Visit
 https://www.drupal.org/node/1897420 for further information.

CONFIGURATION
-------------

* There are no configurations.
